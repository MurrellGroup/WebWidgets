Virus watchpoint map editor

Drag labels. Double-click a label to lock/unlock it. In hotspot mode, click a label to select it; right-click the map to add a hotspot to that label.
Actions and state
View mode: hotspot centroids
In centroid edit mode, drag centroids to move kernels. Mouse-wheel over a centroid changes its intensity. Shift = faster. Alt/Option = finer. Select a hotspot and press Backspace/Delete to remove it.
Regions, boxes, viruses
Box toggles affect annotations. Virus toggles affect each box’s text. Hotspot toggles affect contours.
